# 错误码说明

| 错误码 | 场景 | 当前含义 |
| --- | --- | --- |
| `SYSTEM_001` | 系统级兜底错误 | 未预期异常或未知错误 |
| `CORE_001` | 消息分发 | 发送了未识别的消息类型 |
| `CORE_002` | 运行时通信 | 页面与后台通信失败 |
| `FOCUS_001` | 配置锁定 | 专注进行中尝试修改白名单或配置 |
| `FOCUS_002` | 会话冲突 | 已有进行中的专注会话，又尝试重新开始 |
| `FOCUS_003` | 会话缺失 | 需要活动会话时，当前却没有进行中的专注 |
| `SESSION_001` | 开始专注 | 学习目的为空 |
| `SESSION_002` | 开始专注 | 时长无效或未选择 |
| `SESSION_003` | 开始专注 | 时长超过当前上限 |
| `SETTINGS_001` | 白名单设置 | 域名格式无效 |
| `SETTINGS_002` | 设置保存 | 自定义时长上限配置无效 |
| `SETTINGS_003` | 设置保存 | 主题配置无效 |
| `SETTINGS_004` | 设置保存 | 统计保留天数无效 |
| `SETTINGS_005` | 设置保存 | 冷却时间分钟数无效 |
| `SETTINGS_006` | 白名单设置 | 白名单域名重复 |
| `UNLOCK_001` | 应急解锁 | 未填写解锁原因 |
| `UNLOCK_002` | 应急解锁 | 当前没有可用题目或题目状态丢失 |
| `UNLOCK_003` | 应急解锁 | 没有待确认的解锁结果 |
| `UNLOCK_004` | 应急解锁 | 传入了不支持的解锁结果 |
| `UNLOCK_005` | 应急解锁 | 答题未全部正确，前端应摇晃答题区并高亮错题 |
| `UNLOCK_006` | 应急解锁 | 当前处于冷却时间内 |
| `UNLOCK_007` | 应急解锁 | 提交答案时存在未填写项 |

## 使用方式

1. 后台抛出错误时统一附带错误码。
2. 运行时消息返回失败时，会带上 `code`、`error`、`details`。
3. 前端页面展示错误提示时，默认显示为 `[错误码] 错误说明`。
4. 测试时如果看到错误码，可以直接根据上表定位到对应模块和场景。
