# 学习专注型 Edge 扩展 页面线框与页面清单

## 1. 页面总览

| 页面 | 建议文件 | 入口 | 主要作用 |
| --- | --- | --- | --- |
| 首页弹窗 | `popup.html` | 点击扩展图标 | 查看当前状态与核心操作入口 |
| 开始专注页 | `pages/start-session.html` | 弹窗中的“开始专注” | 输入学习目的并选择时长 |
| 白名单与设置页 | `pages/options.html` | 弹窗中的“查看白名单” | 管理白名单和用户配置 |
| 拦截页 | `pages/block.html` | 非白名单导航重定向 | 提示被拦截并引导解锁 |
| 数据汇总页 | `pages/dashboard.html` | 弹窗中的“查看汇总” | 展示专注、拦截和解锁数据 |
| 应急解锁页 | `pages/unlock.html` | 弹窗或拦截页中的“应急解锁” | 填写原因、答题、执行解锁 |
| 临时放行提醒窗 | `pages/temp-allow-reminder.html` | 选择“临时放行 10 分钟”后自动打开 | 在浏览器一侧展示倒计时提醒 |

## 2. 页面导航关系

```text
popup
├─ 开始专注 -> start-session
├─ 查看白名单 -> options
├─ 查看汇总 -> dashboard
└─ 应急解锁 -> unlock

block
├─ 返回学习网站 -> options / 预设学习站点
└─ 应急解锁 -> unlock
```

## 3. 全局交互规则

1. 所有页面默认中文。
2. 所有正式页面支持亮色和暗色主题。
3. 专注进行中，`options` 页面进入只读状态。
4. 拦截页与解锁页必须能读到当前会话状态。
5. 临时放行期间应显示独立倒计时提醒窗，并在结束后自动关闭。
6. 开发模式快速退出按钮仅在开发环境可见。

## 4. 页面线框

### 4.1 首页弹窗 `popup.html`

**目的**

提供最短路径的状态查看和核心操作入口。

**状态**

1. 未开始
2. 专注中
3. 已结束
4. 开发模式

**线框**

```text
+----------------------------------+
| 学习专注                         |
|----------------------------------|
| 状态：专注中                     |
| 剩余时间：01:28                  |
| 学习目的：完成高数作业第 3 章    |
|----------------------------------|
| [开始专注]                       |
| [查看白名单] [查看汇总]          |
| [应急解锁]                       |
|----------------------------------|
| 开发模式                         |
| [快速退出当前专注]               |
+----------------------------------+
```

**交互说明**

1. 未开始时显示“开始专注”主按钮。
2. 专注中时突出显示剩余时间和学习目的。
3. 开发模式下增加独立调试区，避免与正式功能混淆。

### 4.2 开始专注页 `pages/start-session.html`

**目的**

创建新的专注会话。

**线框**

```text
+------------------------------------------------+
| 开始本次专注                                   |
|------------------------------------------------|
| 学习目的                                       |
| [ 完成英语阅读笔记________________________ ]   |
|                                                |
| 选择时长                                       |
| [30 分钟] [60 分钟] [120 分钟] [180 分钟]      |
|                                                |
| 自定义时长                                     |
| [____] 分钟                                    |
| 当前上限：按用户设置的预设档位生效             |
|                                                |
| 校验提示区域                                   |
|                                                |
| [开始专注]                                     |
+------------------------------------------------+
```

**交互说明**

1. 学习目的为空时，开始按钮禁用或提交时报错。
2. 用户可选快捷时长，也可输入自定义时长。
3. 自定义输入超出当前上限时需即时提示。

### 4.3 白名单与设置页 `pages/options.html`

**目的**

在一个页面中集中处理白名单和用户可配置项。

**页面结构**

1. 顶部页签：`白名单`、`设置`
2. 专注中显示只读横幅
3. 非专注状态下允许保存

**线框**

```text
+------------------------------------------------------+
| 白名单与设置                                         |
|------------------------------------------------------|
| [白名单] [设置]                                      |
|------------------------------------------------------|
| 专注中：当前为只读模式，不能修改配置                 |
|------------------------------------------------------|
| 搜索域名 [____________________]                      |
|                                                      |
| 新增域名 [____________________] [包含子域 v] [添加] |
|                                                      |
| 预设学习站点                                         |
| [wikipedia.org] [coursera.org] [mooc.icourse163.org] |
|                                                      |
| 白名单列表                                           |
| - bilibili.com          包含子域   [删除]            |
| - wikipedia.org         不含子域   [删除]            |
|                                                      |
| [保存更改]                                           |
+------------------------------------------------------+
```

**设置页签建议布局**

```text
+------------------------------------------------------+
| 设置                                                 |
|------------------------------------------------------|
| 自定义时长上限                                       |
| [30-180 分钟 v]                                      |
|                                                      |
| 统计数据保留说明                                     |
| 仅保留最近 7 天，超过一周自动清理                 |
|                                                      |
| 解锁失败冷却                                         |
| [开关]                                               |
| 冷却时间 [____] 分钟                                 |
|                                                      |
| 主题                                                 |
| [亮色] [暗色]                                        |
|                                                      |
| [保存设置]                                           |
+------------------------------------------------------+
```

**交互说明**

1. 专注中所有输入控件禁用。
2. 白名单新增只接受域名，不接受完整 URL。
3. 冷却分钟输入只在开关开启时启用。

### 4.4 拦截页 `pages/block.html`

**目的**

在非白名单访问时给出清晰阻断反馈，并引导回学习路径。

**线框**

```text
+------------------------------------------------+
|                    [锁定图标]                  |
|                                                |
| 当前为学习专注时段，本站不在白名单内           |
|                                                |
| 学习目的：完成操作系统课程实验报告             |
| 剩余时间：52 分钟                              |
| 预计结束：21:30                                |
|                                                |
| [返回学习网站]                                  |
| [应急解锁]                                      |
+------------------------------------------------+
```

**交互说明**

1. 页面需要保持高对比度，减少多余装饰。
2. “返回学习网站”可以跳转到白名单页或最近使用的白名单站点。
3. 拦截页不提供直接结束按钮。

### 4.5 应急解锁页 `pages/unlock.html`

**目的**

为真正需要退出限制的情况提供带成本的解锁通道。

**步骤拆分**

1. 原因填写
2. 答题
3. 结果选择

**线框：原因填写 + 答题**

```text
+------------------------------------------------------+
| 应急解锁                                             |
|------------------------------------------------------|
| 请填写解锁原因                                       |
| [_______________________________________________]    |
|                                                      |
| 请完成以下 5 道两位数乘法题                          |
| 1. 23 x 47 = [____]                                  |
| 2. 68 x 14 = [____]                                  |
| 3. 35 x 26 = [____]                                  |
| 4. 72 x 19 = [____]                                  |
| 5. 44 x 58 = [____]                                  |
|                                                      |
| 冷却中提示：还需等待 04:32                            |
|                                                      |
| [提交答案]                                           |
+------------------------------------------------------+
```

**线框：通过后结果选择**

```text
+----------------------------------------------+
| 全部答对，可以选择以下操作                   |
|----------------------------------------------|
| [临时放行 10 分钟]                           |
| [结束本次专注]                               |
+----------------------------------------------+
```

**交互说明**

1. 冷却中时，题目可隐藏或整体禁用。
2. 只有 5 题全部答对才显示结果选择区域。
3. 提交错误答案后，答题区整体摇晃一次，并仅高亮答错题目的卡片与输入框。
4. 用户继续修改答案后，应清除错题高亮和输入错误态。
5. 临时放行后要在当前会话上显示剩余放行时间。
6. 执行成功后应展示本次解锁类型，并在存在原始页面时提供返回按钮。

### 4.6 临时放行提醒窗 `pages/temp-allow-reminder.html`

**目的**

在临时放行阶段于浏览器一侧持续提醒剩余时间，避免用户忘记当前仍处于受控会话中。

**线框**

```text
+----------------------------------+
| 临时放行                         |
|----------------------------------|
| 剩余 08:42                       |
| 本次目的：复习线性代数           |
| 结束时间：21:18                  |
|                                  |
| 倒计时结束后会自动恢复拦截       |
| [查看数据汇总]                   |
+----------------------------------+
```

**交互说明**

1. 窗体默认以小型弹窗形式出现在浏览器一侧，不抢主窗口焦点。
2. 临时放行倒计时结束后自动关闭，并恢复拦截。
3. 若当前没有活动会话或临时放行已结束，页面应展示结束态并短暂延时关闭。

### 4.7 数据汇总页 `pages/dashboard.html`

**目的**

帮助用户复盘专注行为，并从本地统计中看到模式。

**线框**

```text
+------------------------------------------------------+
| 数据汇总                                             |
|------------------------------------------------------|
| [今日] [本周] [全部]                                 |
|------------------------------------------------------|
| 今日专注总时长   已完成会话数   被拦截次数           |
| 180 分钟         3 次            12 次               |
|------------------------------------------------------|
| 高频被拦截域名                                       |
| 1. youtube.com                                       |
| 2. x.com                                             |
| 3. instagram.com                                     |
|------------------------------------------------------|
| 学习目的记录                                         |
| - 完成高数习题        90 分钟        20:00-21:30     |
| - 阅读论文            60 分钟        18:00-19:00     |
|------------------------------------------------------|
| 解锁记录                                             |
| - 临时放行 10 分钟  原因：回复消息  17:35            |
+------------------------------------------------------+
```

**交互说明**

1. 页面优先展示汇总卡片，再展示明细列表。
2. 历史记录按时间倒序排列。
3. 数据固定保留最近 7 天，超过一周自动清理。

## 5. 开发模式专用交互

### 5.1 快速退出按钮

**建议位置**

1. 弹窗底部独立调试区域
2. 或 `options` 页面中的开发模式卡片

**行为定义**

1. 立即结束当前专注会话
2. 清理动态拦截规则
3. 清理临时放行和冷却状态
4. 记录一条开发模式退出日志

**限制**

1. 仅开发模式显示
2. 正式构建中默认移除
3. UI 文案明确标记“调试专用”

## 6. 页面实现优先级

1. `popup.html`
2. `pages/start-session.html`
3. `pages/options.html`
4. `pages/block.html`
5. `pages/unlock.html`
6. `pages/dashboard.html`


## 7. 视觉补充说明

1. `popup.html`、`pages/start-session.html`、`pages/options.html`、`pages/unlock.html`、`pages/block.html`、`pages/dashboard.html`、`pages/temp-allow-reminder.html` 统一使用暗色极光动态背景。
2. `dashboard` 作为数据汇总页，允许更强的顶部极光光带和更明显的缓慢漂移；其它页面沿用同风格，但强度略低于 `dashboard`。
3. 页面内容容器需保持玻璃质感和高对比可读性，极光层应位于内容后方，避免遮挡正文与操作控件。
4. 若系统偏好减少动态效果，保留暗色渐变与静态光晕，不再执行持续漂移动画。
